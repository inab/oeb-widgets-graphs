OEB Widgets Graphs Docs

Overview

This is a collection of graphs made by OEB services. The graphs are made using the Plotly.js library.

Installation

To install the graphs, you can use the following command: 

npm install oeb-widgets-graphs
Usage
In base javasctipt applications

Add the installed package to your javascript file. 

import '@inb/oeb-widgets-graphs';
Then you can use the graphs in your application. 
<widget-element 
  :data=graphData
  :type=graphType>
</widget-element>

In base HTML applications (without a framework)

Add the installed package to your HTML file. 

<script src="node_modules/oeb-widgets-graphs/dist/oeb-widgets-graphs.js"></script>
Then you can use the graphs in your application. 
<widget-element 
  id="widget-element"
  data=graphData
  type=graphType>
</widget-element>

<script>
  const graphType=JSON.stringify("bar-plot");
  const myComp = document.getElementById('widget-element');
  // Change the type and data of the graph
  let type = JSON.parse(graphType);
  myComp.type = type;
  myComp.data = graphData;
</script>
